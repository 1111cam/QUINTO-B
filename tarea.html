sintaxis de java

;

La sintaxis de Java es el conjunto de reglas que determinan cómo se deben escribir los programas en este lenguaje.

estructuras repetitivas

En Java, las estructuras repetitivas (o bucles) permiten ejecutar un bloque de código varias veces.

estructuras repertitivas

Las estructuras repetitivas (o bucles) son instrucciones que permiten repetir un bloque de código varias veces hasta que se cumpla una condición.

herencia

La herencia en Java es un mecanismo que permite que una clase (hija o subclase) adquiera los atributos y métodos de otra clase (padre o superclase). Es uno de los pilares de la programación orientada a objetos (POO).
